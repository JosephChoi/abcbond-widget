🔌 ABC Bond API 연결 테스트

실제 API 서버와의 연결을 테스트합니다

1. Health Check

엔드포인트: GET /health
설명: API 서버가 정상 작동하는지 확인합니다.

2. 로그인

엔드포인트: POST /auth/login
설명: 사용자 인증을 수행하고 JWT 토큰을 받습니다.

3. 내 투자 내역 조회

엔드포인트: GET /user-investments/my
설명: 로그인한 사용자의 투자 내역을 조회합니다.
인증 필요: ✅ (먼저 로그인해야 합니다)

4. 투자 상품 상세 조회

엔드포인트: GET /investments/:id
설명: 특정 투자 상품의 상세 정보를 조회합니다.

5. 내 프로필 조회

엔드포인트: GET /users/profile
설명: 로그인한 사용자의 프로필 정보를 조회합니다.
인증 필요: ✅

6. 내 투자 통계 조회

엔드포인트: GET /user-investments/my/stats
설명: 총 투자금액, 예상 수익률 등 통계를 조회합니다.
인증 필요: ✅
